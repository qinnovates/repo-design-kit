# Skins — Neo-Y2K Futurism

Liquid glass SVG headers generated by `scripts/gen-header.py`. Each skin defines a light bar gradient, title gradient, glass tint, and glow color for dark and light modes.

```bash
python scripts/gen-header.py "PROJECT" --skin chrome --out docs/assets/
```

---

## Chrome

Metallic silver-to-white. Y2K mirror finish.

```
Dark:   #E0E0E0 → #FFFFFF → #E0E0E0
Light:  #B0B0B0 → #D4D4D4 → #B0B0B0
Badge:  color=E0E0E0
```

Best for: Dev tools, infrastructure, CLI tools, anything that should feel precision-machined.

## Generating Headers

```bash
# Basic
python scripts/gen-header.py "TITLE" --skin chrome

# With subtitle and tagline
python scripts/gen-header.py "TITLE" -s "subtitle" -t "tagline text" --skin hologram

# Custom output directory
python scripts/gen-header.py "TITLE" --skin aurora --out my-project/docs/assets/

# Custom dimensions
python scripts/gen-header.py "TITLE" --skin ember --width 800 --height 220
```

Output: `header-dark.svg` + `header-light.svg` in the output directory.

Reference them in your README:

```markdown
<p align="center">
  <picture>
    <source media="(prefers-color-scheme: dark)" srcset="docs/assets/header-dark.svg">
    <source media="(prefers-color-scheme: light)" srcset="docs/assets/header-light.svg">
    <img alt="TITLE" src="docs/assets/header-dark.svg" width="700">
  </picture>
</p>
```

## Glass Anatomy

Every generated header has the same structure:

```
┌──────────────────────────────────────────┐
│ ═══════ light bar (glowing) ═══════════  │  ← Turrell-inspired luminous strip
│                                          │
│          T I T L E                       │  ← Gradient text with glow filter
│          subtitle                        │  ← Muted, spaced
│     tagline goes here                    │  ← Smallest, most subtle
│                                   KevQ   │  ← 8% opacity watermark
└──────────────────────────────────────────┘
     ↑ rounded glass panel with:
       • frosted glass fill (rgba tint)
       • 1px glass stroke (subtle border)
       • inner highlight gradient (top-bright → center-dim → bottom-slight)
       • light bar glow bleeds through panel
```

No external dependencies. No shields.io. No fonts to load. Pure SVG with filters.
