# Skins — Neo-Y2K Futurism

Pill-shaped glass chrome banners generated by `scripts/gen-header.py`. Pick a skin, pass the name, get dark + light SVGs.

Preview all skins: open `templates/skins/preview/turrell-cloud.html` and `experimental.html` in a browser.

```bash
python scripts/gen-header.py "PROJECT" --skin plasma-vesicle --out docs/assets/
```

## CSS Reference

Every skin has these layers inside the chrome pill shell:

```
┌─────────────────────────────────────────────┐  ← chrome rim (gradient border)
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │  ← specular top arc
│           TURRELL LAYER (radial glow)        │  ← aperture light
│           CLOUD LAYER (atmosphere)           │  ← volumetric fog
│     ════════ BAR (gradient) ════════════     │  ← center line
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │  ← bottom bounce
└─────────────────────────────────────────────┘

  ●  ●  ●     ← blobs behind pill (blurred, frosted through glass)
```

Experimental skins may replace the Turrell/Cloud layers with different techniques (linear splits, multiple radials, noise fields, etc).

## Adding a Custom Skin

Add a new entry to `gen-header.py`'s SKINS dict:

```python
"my-skin": {
    "dark": {
        "turrell": "radial-gradient(...)",
        "cloud": "radial-gradient(...)",
        "bar": ["#hex1", "#hex2", "#hex3"],
        "blobs": [("r,g,b", opacity), ...],
    },
    "light": { ... }
}
```

Or just edit the HTML preview files directly — they're the source of truth for visual iteration.
